# Typing Speed Test

A responsive web application to test and improve your typing speed. This project allows users to practice typing with random quotes, track their accuracy, count mistakes, and measure their typing speed in Words Per Minute (WPM).

## Features

- **Dynamic Quote Generation**: Fetches random quotes from the [Quotable API](https://github.com/lukePeavey/quotable).
- **Real-Time Feedback**:
  - Tracks typing errors.
  - Displays WPM and accuracy in real-time.
- **User-Friendly Interface**: Clean and responsive design for better user experience.
- **Timer Functionality**: Provides a 60-second test duration for a focused typing challenge.

## Usage

1. Click on **Start Test** to begin.
2. Type the displayed quote in the input area.
3. Press **Stop Test** or wait for the timer to end to see your results:
   - **WPM**: Measures your typing speed.
   - **Accuracy**: Tracks your correct inputs.
   - **Mistakes**: Counts your typing errors.
